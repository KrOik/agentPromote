# 论文分析报告模板

以下是输出报告的标准结构和各维度的详细分析指引。

---

## 一、论文综述

### 分析要点

- 梳理Introduction中的文献线索，通常包含2-4条主线
- 识别论文试图解决的核心问题
- 总结现有方法的局限性（即research gap）
- 用简洁语言概括：该领域做了什么，还有什么没做好，本文要补什么

### 写作要求

- 不复述摘要，用自己的话重新组织
- 突出从"背景 -> 现状 -> 不足 -> 本文动机"的逻辑链条
- 引用论文中提到的关键文献作为综述线索

---

## 二、创新点（个人视角）

### 分析要点

- 从以下维度评估创新性：
  - 材料创新（新材料、新结构设计）
  - 方法创新（新工艺、新检测原理）
  - 系统集成创新（多功能融合、新架构）
  - 应用创新（新场景、新闭环）
- 区分"真正的创新"与"增量改进"
- 对每个创新点给出简短的技术评价

### 写作要求

- 使用编号列表，每个创新点包含：是什么 + 为什么新颖 + 技术评价
- 基于自己的专业知识给出独立判断，不照搬论文的claims

---

## 三、文章状态

### 需要确认的信息

- 发表状态：已发表 / 已接受待发表 / 预印本 / 审稿中
- OA类型：Gold OA / Green OA / Bronze OA / 非OA
- 许可协议：CC BY / CC BY-NC-ND / 其他
- APC费用（如有）
- 投稿时间线：收稿日期 / 修回日期 / 接受日期 / 在线发表日期

### 信息来源

- 从PDF首页的出版信息、页眉/页脚、版权声明中提取
- 预印本检查 arxiv.org / biorxiv.org 等

## 五、实验设计

### 分类框架

**干实验（计算/模拟类）：**
- 仿真模拟
- 机器学习建模（注明算法、数据集大小、验证方式）
- 数据库挖掘/生物信息学分析
- 计算化学/有限元分析

**湿实验（物理/化学/生物类）：**
- 材料合成与制备
- 结构与形貌表征（SEM/TEM/XRD/XPS等）
- 电化学/光学/力学性能测试
- 体外实验（细胞、组织）
- 动物实验
- 人体在体实验（注明伦理审查机构、受试者数量、排除标准）

### 写作要求

- 按实验类别分组列出
- 每个实验说明：目的、方法、关键参数

## 七、业界贡献

### 分析维度

- 方法学贡献：是否提出了可推广的新方法/新策略
- 技术贡献：性能指标是否达到或超越state-of-the-art
- 应用贡献：是否打通了从实验室到实际应用的链路
- 平台贡献：是否建立了通用框架/平台，可复用于其他场景
- 数据贡献：是否公开了有价值的数据集

### 写作要求

- 用编号列表列出，每条贡献附简短说明
- 客观评价，不夸大

---

## 八、图例分布

### 分析格式

对每张主图按以下格式描述：

**Fig. N（主题关键词，占X页）**

- (a) 子图描述
- (b) 子图描述
- ...

### 补充图

统计补充图数量（Fig. S1 - Fig. SN），列出补充图中涵盖但主图未展示的内容类型。

### 注意事项

- 每张图的每个子面板（a, b, c...）都必须描述
- 标注标尺信息（如有）
- 说明图中关键数据和趋势

---

## 总结评价模板

一段话总结：
1. 论文的核心贡献（1-2句）
2. 主要创新点的价值判断（1-2句）
3. 实验设计的充分性评价（1句）
4. 主要局限性（1-2句）
5. 期刊档次与国内认可度（1句）
